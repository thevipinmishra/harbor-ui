# Button

Buttons are interactive elements that users can click or tap to trigger an action.

## Usage

```jsx
import { Button } from '@harbor-ui/react'

function Example() {
  return (
    <Button variant="primary">
      Click me
    </Button>
  )
}
```

## Variants

The Button component comes with several predefined variants:

- `primary` - Used for primary actions
- `secondary` - Used for secondary actions
- `outline` - A bordered button with transparent background
- `ghost` - A minimal button with no background or border

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| variant | 'primary' \| 'secondary' \| 'outline' \| 'ghost' | 'primary' | Visual style variant |
| size | 'sm' \| 'md' \| 'lg' | 'md' | Button size |
| disabled | boolean | false | Disables button interactions |
| loading | boolean | false | Shows loading spinner |
| onClick | function | - | Click handler function |
| children | ReactNode | - | Button content |
